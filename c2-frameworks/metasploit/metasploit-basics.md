# Metasploit Basics

### Basics

* Search for modules&#x20;

```
search [regex]
```

* Specify exploit

```
use exploit /path/to/exploit
```

* Specify Payload&#x20;

```
set payload path/to/payload
```

* Set option

```
set [option] [value]
```

* Run Exploit

```
run
exploit 
#either will work 
```
